# Moe's Vocab Specification

## 単語選択ロジック (2026-03-24 更新)

### 基本パラメータ
- **1日の最大単語数 (DAILY_LIMIT)**: 50
- **新規単語の最低保証 (MIN_NEW)**: 10
- **復習枠の最大数**: 40 (DAILY_LIMIT - MIN_NEW)

### 分類ルール
学習済みの全単語を以下の3つのバケットに分類します。

1.  **red (前回不正解)**: 前回の学習で「知らなかった (quality=0)」と判定された単語。
2.  **due (復習対象)**: SM-2アルゴリズムに基づき、今日が復習予定日（lastSeen + interval）を過ぎている単語。
3.  **new (新規)**: まだ一度も学習していない単語。
    - 優先順位: Part降順 (3->2->1) -> Section昇順 -> ID昇順

### 選択アルゴリズム
合計50語を選定する際、以下の優先順位で枠を埋めます。

1.  **復習枠 (最大40語)**:
    - まず **red** バケットから最大40語まで抽出。
    - 40語に満たない場合、残りの枠で **due** バケットから抽出。
2.  **新規枠 (最低10語 + α)**:
    - **MIN_NEW (10語)** は、復習枠の状態に関わらず必ず含める。
    - 復習枠（red + due）が40語に満たなかった場合、余った枠もすべて **new** で埋め、合計50語にする。

#### 例
- red(30), due(20) の場合: red(30) + due(10) + new(10) = 50語
- red(5), due(10) の場合: red(5) + due(10) + new(35) = 50語
- red(45) の場合: red(40) + new(10) = 50語 (5語のredは翌日へ)
